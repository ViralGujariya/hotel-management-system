# 🏨 Hotel Management System

A modern and user-friendly **Hotel Management System** built using **HTML, CSS, JavaScript, and JSON (Local Storage)**. This project allows customers to book rooms easily and provides an **admin panel** to manage reservations efficiently.

---

## 🚀 Features

### 👤 Customer Side

* Book rooms with:

  * Name
  * Date
  * Room Type (Single / Double / Suite)
* Simple and clean booking interface
* Instant booking confirmation

### 🔑 Admin Panel

* View all reservations
* Delete bookings
* Manage customer data easily
* Real-time updates using local storage

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Database:** JSON (LocalStorage)
* **Tools:** VS Code, Live Server

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/hotel-management-system.git
   ```

2. Open project folder:

   ```bash
   cd hotel-management-system
   ```

3. Run using Live Server:

   * Right-click `index.html`
   * Click **Open with Live Server**
